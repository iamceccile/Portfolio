# Portfolio – Frontend

Portfolio personnel développé avec **React, Vite et TailwindCSS**.

Ce projet présente différents projets réalisés durant mon parcours à **Epitech** ainsi que des projets personnels.

---

# Installation

Cloner le repository :

```bash
git clone https://github.com/iamceccile/portfolio.git
cd portfolio
```

Installer les dépendances :

```bash
npm install
```

---

# Lancer le projet

Démarrer le serveur de développement :

```bash
npm run dev
```

Le projet sera accessible sur :

```
http://localhost:5173
```

---

# Build du projet

Pour générer la version de production :

```bash
npm run build
```

Le build sera généré dans le dossier :

```
dist/
```

Pour prévisualiser le build :

```bash
npm run preview
```
